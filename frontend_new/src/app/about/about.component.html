About PDF Editor

This program was written in 2022-2023 for creation and modification of files in PDF format. It enables to insert titles, text and images to document with the use of special tags:

\title - insertion of the title with default font size (12) at horizontal center
\title(fontSize) - insertion of the title with specified font size at horizontal center
\text - insertion of the text with default font size (12)
\text(fontSize) - insertion of the text with specified font size
\img - insertion of the image with default width(300) and height(200) at horizontal center
\img(width,height) - insertion of the image with specified width and height at horizontal center
\\ - comment just for user needs, not inserted

It's required to have a separation space between tag and value. In case of image the user must write the name of the image (with extension). For instance: \img example_image.jpg.

Example usages of tags:

\title(16) Example title
\text Example text
\img(200,150) example_image.png
\\ example comment
